La Foire aux questions du centre d'assistance

Chaque année les élèves de 6ITN instruisent quelques professeurs comment on utilise un programme qui est utilisé à l’école. Sur cette page, vous trouvez des questions fréquemment posées de Microsoft Teams. Malheureusement le IT-tapa ne peut pas se passer cette année mais j’espère que vous apprendrez beaucoup de ce programme utile avec ces questions.

Pourquoi vous devrions utiliser Microsoft Teams ?
Je conseille que tous les professeurs utilisent Microsoft Teams dans tes leçons. Voici quelques raisons. Le programme est très bon pour la collaboration et la communication entre les étudiants et le professeur. Quand un(e) étudiant(e) a une question, le professeur ou les autres étudiants peuvent répondre immédiatement dans la fenêtre de chat. C’est très rapide et efficace. C’est aussi utile pour donner des tests et partager du matériel pédagogique.
Quelle est la meilleure façon de donner un test à mes élèves et pourquoi ?
Moi, je conseille un quiz de teams que les élèves peuvent résoudre. C’est une jolie façon pour faire un test et c’est très utile et efface pour vous aussi. Vous pouvez ajouter les réponses correctement avant que vous distribuiez le quiz. Ça le rend plus facile et rapide pour le corriger. Vous pouvez aussi sauvegarder le quiz et l’utiliser dans les autres classes.
Comment fait-on un quiz dans Teams ?
Dans le menu des devoirs de votre équipe, cliquez sur le bouton ‘Créer’ et après cliquez sur ‘Questionnaire’. Ici vous venez dans un menu où vous pouvez choisir un quiz que vous avez déjà fait ou vous pouvez faire un nouveau quiz. Pour faire un nouveau quiz, cliquez sur ‘+ nouveau formulaire’. Ça vous conduit à Microsoft Forms. Ici vous pouvez entrer le nom du quiz et ajouter différentes sortes de questions et vos bonnes réponses. Vous pouvez aussi ajouter le nombre de points d’un question. Quand votre quiz est fini, vous pouvez fermer Microsoft Forms. Le quiz sauvegarde automatiquement.
Comment distribue-on un quiz dans Microsoft Teams ?
Pour distribuer ton quiz avec tes élèves, cliquez sur le bouton ‘Créer’ chez le menu des devoirs de ta équipe et après cliquez sur ‘Questionnaire’. Ici choisissez le quiz qui les élèves doivent résoudre et cliquez sur ‘suivant’. Voici un menu. Ici on peut entrer un instruction et modifier l’heure d’échéance. Quand vous avez fait ça, vous pouvez cliquer sur le bouton ‘affecter’ et les élèves de ta équipe peuvent résoudre le quiz. Quand tous élèves ont retourné le quiz, je conseils de corriger question par question et pas élève par élève. Microsoft Forms a un système utile avec lequel vous pouvez voir les réponses de tous les élèves dans une page et les corriger. C’est une façon plus efficace.
